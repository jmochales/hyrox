# Planificación y Seguimiento de Entrenamientos

## Filosofía

La preparación para Hyrox requiere un equilibrio entre:
- **Carrera** (8 km totales en competición, en tramos de 1 km)
- **Fuerza/resistencia funcional** (8 estaciones de trabajo)
- **Capacidad aeróbica** para mantener rendimiento durante 60-90 min

## Distribución semanal recomendada

### Semana tipo (4-5 sesiones)

| Día | Tipo | Foco |
|-----|------|------|
| Lunes | Fuerza | Tren superior + core |
| Martes | Carrera intervalos | Velocidad / VO2max |
| Miércoles | Descanso / movilidad | Recuperación |
| Jueves | Híbrido | Simulación Hyrox |
| Viernes | Fuerza | Tren inferior + funcional |
| Sábado | Carrera larga | Fondo aeróbico |
| Domingo | Descanso | Recuperación |

### Semana tipo (3 sesiones - mantenimiento)

| Día | Tipo | Foco |
|-----|------|------|
| Martes | Híbrido | Simulación Hyrox |
| Jueves | Fuerza | Full body |
| Sábado | Carrera larga | Fondo aeróbico |

## Cómo usar el plan

1. Definir tu plan en `plan/current.md` indicando:
   - Objetivo (fecha de competición, tiempo objetivo)
   - Semanas de preparación
   - Distribución semanal elegida

2. Cada semana, la skill puede:
   - Comparar entrenamientos realizados vs planificados
   - Detectar semanas con bajo volumen
   - Sugerir ajustes si hay desequilibrios

3. Al finalizar un ciclo, mover `plan/current.md` a `plan/archive/` y crear uno nuevo.

## Seguimiento

### Métricas clave para Hyrox

| Métrica | Cómo medirla | Objetivo |
|---------|--------------|----------|
| Ritmo carrera 1km | Carreras intervalos | < 5:00/km |
| SkiErg 1000m | Entrenamientos híbridos | < 4:00 |
| Sled Push 50m | Entrenamientos fuerza/híbrido | < 2:00 |
| Wall Balls x100 | Entrenamientos fuerza/híbrido | < 5:00 |
| Tiempo total simulación | Entrenamientos híbridos | Reducir progresivamente |

### Indicadores de carga

- **Volumen semanal**: Tiempo total de trabajo
- **Intensidad media**: RPE medio de la semana
- **Ratio trabajo/descanso**: Semanas de carga vs descarga (3:1 recomendado)
